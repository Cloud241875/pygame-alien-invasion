# 简要介绍
## 游戏介绍
### 玩家将驾驶一艘宇宙飞船，阻挡外星人的入侵，每消灭一群向下移动的外星人后，玩家都将提高一个等级；而等级越高，游戏的节奏越快，难度越大。
## 制作由来
### 这是一个人尽皆知的小游戏，出自《python:从入门到精通》一书，我通过制作这个游戏来练习Python，对git版本管理的使用，还有提升github的使用熟练度
## 想说的话
